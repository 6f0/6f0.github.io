Previously I had been able to pull down the VirtualBox rpm, install and go. Lately however, I've been running into trouble with the kernel modules. These are the steps I took to get around the issue, I did this a while ago and am writing this from memory so if I miss something let me know in the comments and I'll make the updates.

Swith to root User

The following steps will require root permissions.

sudo su -

Grab the Oracle public key and import it

wget -q https://www.virtualbox.org/download/oracle_vbox.asc -O- | rpm --import -

Pull down the repo file

cd /etc/yum.repos.d/ && wget https://download.virtualbox.org/virtualbox/rpm/fedora/virtualbox.repo

Update the package list

dnf update

Install VirtualBox

dnf install VirtualBox-5.2 If you need to compile the kernel modules

Install the necessary packages

dnf install binutils gcc make patch libgomp glibc-headers glibc-devel kernel-headers kernel-devel dkms

Build/rebuild the kernel modules

/usr/lib/virtualbox/vboxdrv.sh setup

After following these steps you should have a working VirtualBox install.
