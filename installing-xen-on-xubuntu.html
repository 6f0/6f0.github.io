Installing Xen on Xubuntu

After following this tutorial on installing Xen on Ubuntu I ran into this little number:

ERROR: Can't find hypervisor information in sysfs!

After scouring the Internets, I found that installing a few more packages and manually adding things to the the end of the xend-config.sxp all is well.

first: install some more shit

sudo apt-get install xen-utils xenwatch xen-tools xen-utils-common xenstore-utils virtinst virt-viewer virt-manager

Next open up the xend-config.sxp file and append this at the end:

sudo nano /etc/xen/xend-config.sxp

(xend-http-server yes)
(xend-unix-server yes)
(vif-script vif-bridge)

Save and close.

Carry on with updating the boot order and...

After these few tweaks you should be able to reboot and carry on.
